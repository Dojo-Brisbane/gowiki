You can download all pages in Markdown either by clicking the on View All page or curl -L -k ${HOST}/pages/download -o ${FILE}.tar.gz in your terminal.

Check it out by using curl -k ${HOST}/pages/search/raw/?q=${SEARCHQUERY}&s=${SIZE} in your terminal. You can also modify the trim size by appending &s=${SIZE}

Spoiler text. Note that it's important to have a space after the summary tag.
You should be able to write any markdown you want inside the details tag... just make sure you close details afterward.
